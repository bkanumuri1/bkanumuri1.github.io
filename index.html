Recent projects

Senior Fall Prevention

IoT and Backend Development

Empower senior citizens in assisted living to maintain their independence while ensuring timely assistance. Integrated motion, heart rate, and pressure sensors, combined with an Alexa-enabled smart lighting system, automatically illuminate the room when a resident gets out of bed. If no motion is detected or a pathway is not completed, a three-level alert system notifies caregivers or security to ensure prompt attention and safety. Project Link YouTube.

GitView

Fullstack Development

An intuitive web application that translates intricate GitHub repository data into user-friendly visualizations. This tool was developed to gain insights about agile software engineering projects. This tool can be utilized by product owners/scrum masters to keep track of teammates' progress from the beginning of the sprint until the end of the sprint. Some metrics that are presented are # of commits made, pull-requests raised, branch merges, comments made on each PR, and any code reviews done on each PR. Project Link YouTube.

Scrum Metrics

Web Application

A distributed web application with microservices architecture that calculates various scrum metrics of agile software projects that are publically available on Taiga. This was made as a Scrum-of-Scrum model of 18 developers at Arizona State University. Project Link YouTube

Soccer Tournament

Product Design

A responsive web app for scheduling and managing games for a soccer tournament. We implemented a round-robin algorithm to auto-schedule upto 850 teams in the tournament. Specialized user logins enable super users such as Tournament Director, Team Director, Referee Director, Field Director to customize details of the tournament, post results and make announcements.
